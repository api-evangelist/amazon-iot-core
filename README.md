# Amazon IoT Core

Amazon IoT Core is a managed cloud service that lets connected devices easily and securely interact with cloud applications and other devices, supporting billions of devices and trillions of messages with reliable and secure communication.

## APIs

- **Amazon IoT Core API** - The Amazon IoT Core API provides programmatic access to manage IoT things, certificates, policies, rules, device shadows, and MQTT message broker configurations for connecting and managing IoT devices at scale.
